# DiabloIIToolbox

暗黑2工具箱 自用 执行一些箱子整理归类 自动登录之类的操作

<hr>暂定<hr>

## 环境

ahk自动化
python图像识别(ahk的图像识别有点慢,之后有空换成直接游戏内存读背包)
vue前端图形界面

- [x] 前端调用ahk函数

### 自动登录

依照文件配置设定账号和密码

```
autoLogin.ini

[%起个名字%]
;[必要]账号
account=账号
;[必要]密码
password=密码
;第几位角色 0不自动选角色 从左往右从上到下1-8
charIndex=第几位角色
;国度 0不动
world=0
```

```
[bo寻蛮子]
;[必要]账号
account=abc003
;[必要]密码
password=123456
;第几位角色 0不自动选角色 从左往右从上到下1-8
charIndex=5
;国度 0默认
world=0
```

### 自定义启动参数

定义多组启动参数

个人用法:用来多个角色使用不同配置的hackmap
其他角色拾取到盒子里,丢东西直接丢盒子就是一键全丢
仓库号hm自动拾取设置捡到包里,不用去盒子里拿出来再进仓库
rr小号设置自动进门,大号不自动进.
波寻蛮子设掉落全不可见,左键绑个寻挂个跟随,不用担心鼠标碰到掉落物

兄弟们自由发挥

### 自动转移 AtoB

暂时只考虑宝石和符文
A为需要转移材料的号
B为仓库号 hm设置自动拾取到背包
因为是一个一个丢+自动拾取 所以丢盒子里的东西铺一地太远捡不到的情况不考虑

```
流程:
1. A打开仓库
2  B站A边上(不打开任何界面)
3. 开始执行
4. A从仓库和背包找对应道具丢地上 默认翻找所有仓库页
5. B自动拾取
6. B自动开仓库
7. 参照设定规则存入
8. 回到第4步 直到A仓库+背包找不到对应道具/B规则无存放空间
```

```
cacheRule.ini
存放规则:
有物品的格子会被忽略

[%规则名字%]
;key翻阅itemSet.ini
key=value

;单格指定 完美的蓝宝石将被存放在第五个格子内
完美的蓝宝石=5

;多格指定 7号符文将被存放在8,6,9三个格子内
符文7=8,6,9

;范围指定 3号符文将被存放在第一到二十格子内
符文33=1-20

;混用
符文1=30-35,18,40-60,12,15

;特殊指定all 全格子可存符文1
符文1=all

;特殊指定empty 第一格留空
empty=1

```
